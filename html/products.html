IoT Platform

The IoT Platform is a very flexible software that supports fast device data collection and processing development. The core consists of a device gateway and a robust messaging bus. Other capabilities are build in micro services and deployed in a Docker Swarm network. It supports various communication protocols, including MQTT, HTTP, TCP/UDP and Modbus.
