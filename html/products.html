This MES product is designed specifically for the precision finishing manufacturers. IoT Platform

The IoT Platform is a flexible framework supporting fast application development related to device data collecting and processing. The core consists of a device gateway and a robust messaging bus. Additional features reside in micro services that are deployed in a Docker Swarm network. It supports various communication protocols, including MQTT, HTTP, TCP/UDP and Modbus.
