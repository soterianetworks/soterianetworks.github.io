IoT Platform


This IoT Platform has been used in multiple IoT projects. Use cases includes humidity, temperature, electric current, vehicle tracking and so on.

  • Device Gateway is the data entry point where different communication protocols are used to extract data from incoming data stream.
  • Messing Bus is the data exchange point. Data is processed by micro services and results are persisted, or published to the Message Bus for other components to use, such as websocket enabled UI pages.
