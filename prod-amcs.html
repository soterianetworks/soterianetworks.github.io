Vision Gauge

This portable vision gauge is a very versatile optical device with foreground and background lighting control. Shipped with our visioning software, it supports bar code and QR code scan, OCR, precision CMM inspection, and defect detection. The compact size allows it to be easily integrated with a robotic arm at very affordable price.

Features

  • Charactertics:
    • Foreground and background lighting switch
    • Horizontal & vertical operation
    • Versatile and economical
    • Easy handling and integration
  • Components:
    • Industry Camera (1)
    • Optical Lens (1)
    • Lighting(2) & Lighting Control
    • Gigabyte Ethernet

Specifications

  • Hardware:
    • Dimensions: 400mm x 150mm x 150mm
    • Weight: 5kg
    • Rated Power: 10W
  • Performance:
    • Accuracy: < 10um
    • Repeat Precision: 2um
    • Cycle Time: < 1 sec
Applications

  • Coordinated Machine Measurement:
    • Size Measurement
    • Uniformity Checking
  • Defect Detection:
    • Missing Parts
    • Abnormal Objects
  • Others:
    • Object character recognition (OCR)
    • Barcode reading
    • QR code reading
